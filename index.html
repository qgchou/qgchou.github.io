使用TeamViewer

发表于 2018-06-09 | | 阅读次数:

工作后就开始用TeamViewer，因为下班后有时还要控制自己的公司办公电脑，有些工作场景需要24小时监控程序，因此TeamViewer对于职场的人来说，几乎是人人必备，这也的确给自己的工作带来了许多便利。虽然公司配了笔记本，下班后也不用带电脑回家了，在外面吃饭时，有些简单的操作可以通过手机TeamViewer连自己的工作机器来进行，极大地方便了自己的生活。
最近遇到了糟心事，TeamViewer被检测为商用，当然这并没有冤枉我：

“hewlett-packard company”这是什么鬼？

我这才想起自己的笔记本是惠普的，公司向惠普批量采购的？没懂。然后我在另一个文章里看到说Teamviewer账号在累计5台机器上登录会被设定为商用？怎么解决？网上的教程说要改mac地址，我自己的笔记本上改了一通，没生效。知乎上一搜，近期有不少同学跟我遇到的情况一样，被检测为商用。反正我是没找到破解办法，买个许可证呢：

有点小贵，平时大部分时间待在公司，真正用的时间并不多，但远程控制软件就算备套螺丝刀，清理电脑时缺不得，但平时基本是搁在那。

最后我选择弃权。不用TeamViewer，但这需求一直在呢，向日葵了解一下？

学习oracle

发表于 2018-06-06 | | 阅读次数:

mysql中指定返回结果的条数可以用limit，oracle中没有limit，oracle中对等的做法是使用虚拟列rownum来查询结果中的前N行数据。

如要查询emp表中的前5行数据，可用如下语句： 

select * from emp where rownum<=5;

zrange

ZRANGE key start stop [WITHSCORES]
返回有序集key中，指定区间内的成员。可以通过使用WITHSCORES选项，来让成员和它的score值一并返回

redis > ZRANGE salary 0 -1 WITHSCORES  # 显示整个有序集成员

3) "tom"
